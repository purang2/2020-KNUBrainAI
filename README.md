## 학부연수생 활동 개요

**소속/담당 교수님: 경북대학교 IT대학 전자공학부 뇌 인공지능 연구실/ 안상태 교수님**

**연구 분야: 뇌인공지능, 딥러닝, 머신러닝, 자연어처리(NLP)**


- [경북대학교 뇌 인공지능 연구실](https://knu-brainai.github.io/)에서 학부연수생 활동을 하면서 학부연수생끼리 팀을 꾸림

- 딥러닝의 떠오르는 화두인 자연어 처리(NLP, Natural Language Processing) 영역의 딥러닝 모델들을 연구하고 학습하여 최신 모델을 오픈소스로 가져오고 Fine-Tuning을 통해서 챗봇을 개발하고 있음

- [GISTxLG전자 주관 DREAM AI 대회](https://dreamai.kr/fair)에 참가하여 오픈소스를 적절히 분석하고 좋은 모델을 이용하고 전처리 활동에 초점을 두어 챗봇을 만들려고 고군분투 하고 있다. **[팀 '크누아이'/전자공학부 이은찬,박영훈,권현지]**

## 연구 목표

딥러닝 기반의 자연어처리를 이해하고자 노력하는 우리 학부연구생 팀의 최종 목표는 질문을 받았을때 스스로 적절한 대화를 생성해내는 '챗봇'을 만드는 것이다. 그래서 먼저, 딥러닝이 어떻게 동작하는지 공부를 하였고  자연어처리(Natural Language Processing, NLP) 분야의 딥러닝 메커니즘을 공부하고 있다.

## 학부연수생 활동을 하면서 배운 것들 

- Git / Github를 이용하는 방법! : 
큰 회사에 프로그래머가 된다면 정말 유용할 것들이라고 느꼈다. 대학교에서는 이렇게 효율적인 프로젝트를 접하기 쉽지 않기 때문이다. 

- Python 머신러닝/딥러닝 라이브러리: 
교수님의 제자(?)로 들어오기 전에는 머신러닝/딥러닝과 관련 없는 삶을 살 줄만 알았고 담을 쌓았었지만, 막상 접해보니 코드는 몇줄 안되어 너무 쉬운거 아냐? 싶지만 볼수록 어려운 친구라는 점을 느꼈다. 그만큼 매력적인 것 같고 익숙해진 지금은 유명한 책 등의 코드를 따라하면서 많이 이해하고 있는것 같아서 흡족하다. Pandas, Numpy 등등 뿐만 아니라 Tensorflow, Scikit-learn 등등 다양한 라이브러리들과도 다소 친해지게 되었다.

- 서버 사용법: 
서버 사용은 우리 팀원 한명이 주로 잘 다루고 사용을 하고 있기에 개인적으로는 거의 사용을 못 해봤고,  이런것도 있구나 싶은 점이다. 그렇지만 그정도로도 충분히 좋은 지식을 알아가고 자극이 되고 있는 것 같다. 나중에 회사에 가면 접할 수도 있을 것 같다.

- 딥러닝 자연어 처리: 
딥러닝을 조금 공부하고 바로 자연어 처리에 대한 내용들을 교수님께 잘 배울 수 있었다. 때문에 높은 수준의 자연어처리 딥러닝 알고리즘들의 특징을 알게 되었지만, 굉장히 어려운 분야이고 개인적으로 아직 익숙하지 않은 분야인 것 같다. 그러나 계속 발전하고 있는 분야고 활용 분야가 너무나 뚜렷하기 때문에 계속 공부하고 싶은 매력적인 분야인 것 같다.
