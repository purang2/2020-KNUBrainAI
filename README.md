## 학부연수생 활동을 하면서 배운 것들 

### +)느끼고 있는 점들

- Git / Github를 이용하는 방법 : 
깃허브를 만들어야 겠다고 계속 말하고 다녔는데 교수님 덕분에 이 깃허브를 만들 수 있었다. Git도 배울수 있었고, 이 두 가지는 큰 회사에 프로그래머가 된다면 정말 유용할 것들이라고 느꼈다. 대학교에서는 이렇게 효율적인 프로젝트를 접하기 쉽지 않기 때문이다. 

- Python 머신러닝/딥러닝 라이브러리: 
교수님의 제자(?)로 들어오기 전에는 머신러닝/딥러닝과 관련 없는 삶을 살 줄만 알았고 담을 쌓았었지만, 막상 접해보니 코드는 몇줄 안되어 너무 쉬운거 아냐? 싶지만 볼수록 어려운 친구라는 점을 느꼈다. 그만큼 매력적인 것 같고 익숙해진 지금은 유명한 책 등의 코드를 따라하면서 많이 이해하고 있는것 같아서 흡족하다. Pandas, Numpy 등등 뿐만 아니라 Tensorflow, Scikit-learn 등등 다양한 라이브러리들과도 다소 친해지게 되었다.

- 서버 사용법: 
서버 사용은 우리 팀원 한명이 주로 잘 다루고 사용을 하고 있기에 개인적으로는 거의 사용을 못 해봤고,  이런것도 있구나 싶은 점이다. 그렇지만 그정도로도 충분히 좋은 지식을 알아가고 자극이 되고 있는 것 같다. 나중에 회사에 가면 접할 수도 있을 것 같다.

- 딥러닝 자연어 처리: 
딥러닝을 조금 공부하고 바로 자연어 처리에 대한 내용들을 교수님께 잘 배울 수 있었다. 때문에 높은 수준의 자연어처리 딥러닝 알고리즘들의 특징을 알게 되었지만, 굉장히 어려운 분야이고 개인적으로 아직 익숙하지 않은 분야인 것 같다. 그러나 계속 발전하고 있는 분야고 활용 분야가 너무나 뚜렷하기 때문에 계속 공부하고 싶은 매력적인 분야인 것 같다.

**딥러닝과 인공지능에 관심을 가지게 된 계기:**
